# 🚀 Пълен одит на сайта (Full Audit)

> Този workflow изпълнява всички audit агенти в текущата сесия.
> Следвай стъпките последователно — не са нужни отделни команди.
> При всяка фаза изпълни задачата изцяло и запиши резултатите преди да продължиш.

## Ред на изпълнение

### Фаза 1: Анализ (без промени по кода)

1. **Код одит** (вж. `/code-reviewer` за пълни инструкции)
   - Прочети source файловете в `js/`, `styles.css`, `index.html`
   - Идентифицирай дублиран код, memory leaks, XSS рискове, архитектурни проблеми
   - Запиши резултатите в artifact `code_review.md`

2. **Бъг хънт** (вж. `/bug-hunter` за пълни инструкции)
   - Анализирай source файловете в `js/` за null reference грешки, race conditions, z-index конфликти
   - Тествай кошница, продуктов модал, търсене, мобилно меню, филтри в браузъра
   - Запиши резултатите в artifact `bug_report.md`

3. **Performance & SEO одит** (вж. `/performance-seo` за пълни инструкции)
   - Измери размерите на файловете, анализирай CSS/JS/HTML, провери SEO и PWA
   - Запиши резултатите в artifact `performance_seo_report.md`

4. **UX одит** (вж. `/ux-auditor` за пълни инструкции)
   - Направи визуален одит на всички ключови екрани (desktop + mobile)
   - Тествай purchase flow, search flow, mobile navigation
   - Запиши резултатите в artifact `ux_audit.md`

### Фаза 2: Идеи

5. **Генерирай идеи** (вж. `/idea-generator` за пълни инструкции)
   - Базирай идеите на откритията от Фаза 1
   - Категоризирай по сложност: Quick Wins / Средни / Големи
   - Запиши резултатите в artifact `ideas_report.md`

### Фаза 3: Обобщение

6. **Създай Master Report** — artifact `full_audit_report.md`:

```markdown

| Бъгове    | X/10   | N        | N     | N         |
| Perf/SEO  | X/10   | N        | N     | N         |
| UX/UI     | X/10   | N        | N     | N         |

## 🎯 Топ 10 приоритетни задачи
1. ...
2. ...

## 📋 Пълен action plan
...
```

### Фаза 4: Одобрение

7. **Поискай feedback от потребителя**
   - Представи `full_audit_report.md`
   - Попитай кои задачи да се имплементират първо
   - **⛔ Спри тук — изчакай одобрение преди да продължиш с Фаза 5**

### Фаза 5: Имплементация (само след изрично одобрение)

8. **Имплементирай одобрените задачи** (вж. `/implementor` за пълни инструкции)
   - Редактирай само source файловете в `js/`, `styles.css`, `index.html`
   - НЕ редактирай `app.js` директно — rebuild с `node build.js` след всяка промяна
   - Тествай всяка промяна в браузъра

9. **Пусни тестовете** (вж. `/test-runner` за пълни инструкции)
   - Изпълни `npm test` — всички тестове трябва да минават
